# Hacker News
------------

## Description 🤙
-  A News app which is fetching data from Hacker News API. Added a search bar to search the News within the app.
------------

### Host Link 🔗
- https://newsapp-three-pi.vercel.app/

------------

### Key-Takeaways 🔑
- React
- CSS custom properties
- Routes
- React Hooks
- News API
